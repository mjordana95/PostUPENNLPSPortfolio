Lost Pet Finder Application

The Lost Pet Finder caters to pet owners who have lost their pets. Individuals that find an abandoned pet can go to the site and submit their pet’s last known location by placing a pin on the map. Owners post in hopes that others will reach out with good news. Two APIs—OpenWeatherMap and Google Maps—were used to create a dynamic web application. We leveraged Google Maps to display pins with info windows containing the user submitted information, along with the weather information pulled from OpenWeatherMap based on that location.

MSJ Experience-Event Planning Application

The MSJ Experience was created based on the need in the industry. A close family member is an event planner and I noticed that she was keeping all her upcoming events in an archaic way. My group decided to take the out of date way of planning and create an application that was smart, efficient and time saving for an event planner who has multiple events going on. Passport was used for authentication. React was used to design the front end of the application and Express was used to design the back end. MongoDB was used for the database to create the to do list and based on the completion of the tasks, the list will be updated. The application was deployed on Heroku and the files can be found in Github on the link below.
